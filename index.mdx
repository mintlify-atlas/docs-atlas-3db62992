MMDetection is an open-source object detection toolbox built on PyTorch and part of the OpenMMLab project. It provides a rich collection of object detection, instance segmentation, and panoptic segmentation algorithms — from classic two-stage detectors like Faster R-CNN to modern transformer-based models like DINO and Grounding DINO — all in a modular, extensible framework.

## What MMDetection supports

MMDetection covers the full spectrum of modern detection tasks and model architectures:

<CardGroup cols={2}>
  <Card title="Object Detection" icon="crosshairs">
    Two-stage (Faster R-CNN, Cascade R-CNN), one-stage (RetinaNet, FCOS, RTMDet), and transformer-based (DETR, DINO, DAB-DETR) detectors.
  </Card>
  <Card title="Instance Segmentation" icon="layer-group">
    Mask R-CNN, Cascade Mask R-CNN, SOLOv2, CondInst, Mask2Former, and more with pixel-level instance masks.
  </Card>
  <Card title="Panoptic Segmentation" icon="object-group">
    Panoptic FPN, MaskFormer, and Mask2Former for unified semantic and instance segmentation.
  </Card>
  <Card title="Open-Vocabulary Detection" icon="magnifying-glass">
    Grounding DINO and GLIP for language-guided detection using free-form text prompts.

## Get started

<Steps>
  <Step title="Install MMDetection">
    Install PyTorch, then install MMDetection via pip or from source.
    ```bash
    pip install -U openmim
    mim install mmengine
    mim install "mmcv>=2.0.0"
    mim install mmdet
    ```
  </Step>
  <Step title="Download a pretrained model">
    Use the `mim download` command to fetch a config and checkpoint.
    ```bash
    mim download mmdet --config rtmdet_tiny_8xb32-300e_coco --dest .
    ```
  </Step>
  <Step title="Run inference">
    Use the Python API to run detection on an image.
    ```python
    from mmdet.apis import DetInferencer

    inferencer = DetInferencer('rtmdet_tiny_8xb32-300e_coco')
    result = inferencer('demo/demo.jpg', show=True)
    ```
  </Step>
</Steps>
